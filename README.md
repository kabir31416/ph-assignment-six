# 🛠️ DigiTools

**DigiTools** is a sleek, modern Single Page Application (SPA) designed for the digital economy. It serves as a comprehensive marketplace for AI tools, software subscriptions, and digital assets, offering a seamless shopping experience.

---

## 🚀 Features

* **Dynamic Marketplace:** A fast and responsive product catalog powered by structured JSON data, ensuring quick load times and easy updates.
* **Modern UI/UX:** A polished, "app-like" interface built with DaisyUI, featuring a fully responsive layout that looks great on any device.
* **User Feedback System:** Integrated toast notifications via `React-Toastify` to provide instant, beautiful visual cues for user interactions.

---

## 💻 Technologies Used

| Technology | Purpose |
| :--- | :--- |
| **React.js** | Core Library / SPA Architecture |
| **Tailwind CSS** | Utility-first Styling |
| **DaisyUI** | UI Component Library |
| **JavaScript (ES6+)** | Logic & Functional Components |
| **React-Toastify** | Notification System |
| **JSON** | Local Data Management |
